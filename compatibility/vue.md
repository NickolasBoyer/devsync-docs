# Vue JS
Fully functional and tested with modern set-up's.

When creating a Vue.js project with the Vue CLI you should enable CSS sourcemaps in the `vue.config.js` file by setting the `css.sourceMap` to `true`:

``` js
// vue.config.js
module.exports = {
  css: {
    sourceMap: true
  }
}
```
